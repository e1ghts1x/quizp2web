HyperText Markup Language

Tópicos importântes:

  • É uma das linguagens que o navegador consegue interpretar para exibição de conteúdo, composto por elementos hierarquicamente organizados.
  • Um documento HTML válido, precisa seguir a seguinte estrutura abaixo:
  • DOCTYPE: não é uma tag, mas uma instrução que tem como objetivo indicar para o navegador qual a versão do HTML deve ser utilizada na página.
  • Lang: atributo que serve para indicar o idioma do documento, sendo de grande importância informá-lo para que a ferramenta de busca consiga indexar o site corretamente, e também para tornar o site acessível.
  • Head: representa o cabeçalho do documento, contendo metainformações de interesse do navegador (pois não são exibidas na página). Além disso, é necessário inserir a tag Titledentro do head para especificar o título do documento.

Cascading Style Sheets

Tópicos importântes:

  • Linguagem de estilo utilizada para descrever a apresentação de um documento.

Todo elemento HTML, no corpo de uma página Web, está contida em um box (box model). O mesmo é composto por conteúdo (content), margem interna (padding), borda (border) e margem externa (margin).

Logo, a largura de uma box é compostas pelas margens (direita e esquerda) externas e internas, bordas e pela largura do conteúdo. A altura de um box é composta pelas margens (superior e inferior) externas e internas, bordas e pela largura do conteúdo.

Seletores: são responsáveis por determinar para quais elementos a regra de formatação deve ser aplicada. Podemos por meio deles definir estilos para tudo em uma página. São divididos em três categorias:

  • Seletores de elemento: a regra é aplicada para os elementos que adicionamos (p, h1, body, form, list...)
  • Seletores de classe: é um seletor cujo o nome você "inventa" e assim pode ser aplicado a qualquer elemento HTML. Sua sintaxe consiste em um nome (nome da classe) + um ponto
  • Seletores de ID: é um indentificador individual associoado a apenas um elemento no documento, não podendo haver mais de um elemento ligado ao mesmo ID. A sintaxe para esse seletor é o caractere "#" seguido do ID.

Cores: no CSS há diversas formas de escolher uma cor, como por exemplo pelo nome (a especificação da linguagem CSS possui 147 cores diferentes nomeadas) ou por valores em hexadecimal como exemplifica a tabela abaixo

#ffffff white

#00ff00 green

JavaScript

Tópicos importântes:

  • É uma linguagem de programação orientada a objetos, sendo interpretada e executada pelo navegador web.

Variáveis:

As variáveis em JS não possuem um tipo de dados no momento de sua declaração, mas assumem o tipo de dados que estão armazenados naquele instante.

Estruturas de controle:

A linguagem implementa estruturas para controle de fluxo de programa. São elas:

  • If: permite definir a execução ou não de determinado bloco de código de acordo com a condição especificada.
  • Switch: de acordo com o valor da variável testada pelo comando, é determinado o bloco de código que será executado.
  • For: permite a repetição caso a condição especificada seja verdadeira.
  • While: executa a repetição de um bloco de programa (semelhante ao For).
  • Do-while: a verificação da condição ocorre apenas após a execução do bloco de programa a ser repetido, que sempre será executado pelo menos uma vez.
